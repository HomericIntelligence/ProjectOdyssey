# 11-26-2025: Anthropic surprises


## Anthropic Surprises

There were three surprises this week. One I was expecting, the other two I was not.


### Surprise 1 - Opus 4.5

I was expecting Opus to be released soon, and the reason was that Opus 4 was released 8 days after Gemini 2.5 Pro. We already had Sonnet 4.5, but no Opus 4.5, and the models most likely are both trained about the same time, so releasing them at the same time seems like a business decision and not a technical one.

### Surprise 2 - Advanced Tool Calling

Anthropic Engineering had a great article published on 11/24/2025 when Claude 4.5 Opus was annoucned on [Programatic Tool Calling](https://www.anthropic.com/engineering/advanced-tool-use) that validates a tool I am working on. Using agents extensively shows their flaws even with advanced prompts. I've talked about it [before](11-21-2025.md) that the agents will say they run tests, but don't actually run them. So the tool i was building would validate that the commands are run and reported back to the calling agent instead of the sub-agent lying about it. This is what they are calling Programatic Tool Calling.

### Surprise 3 - Agentic Harness

The second surprise was an article that came out on 11/26/2025 about having a [harness for long running agents](https://www.anthropic.com/engineering/effective-harnesses-for-long-running-agents). This is very similar to how I am storing updated notes and works in the github issues README.md file and in the github issues themselves. This allows various agents to read important information about what was accomplished by looking at the related github issuse to the one that was already done.
