# Papers

This directory contains implementations of classic machine learning research papers in Mojo.

## Overview

Each paper implementation is organized in its own subdirectory with:

- Source code (`src/`) - Mojo implementation of the paper's models and algorithms
- Tests (`tests/`) - Comprehensive test suite
- Documentation (`README.md`) - Paper details, implementation notes, and usage
- Examples (`examples/`) - Demonstration scripts and notebooks

## Structure

```text
papers/
├── <paper-name>/
│   ├── src/              # Implementation code
│   ├── scripts/          # Scripts to download/setup paper dependencies
│   ├── tests/            # Test suite
│   ├── examples/         # Usage examples
│   └── README.md         # Paper-specific documentation
└── README.md             # This file
```text

When implementing a new paper:

1. Create a new subdirectory with a descriptive name (e.g., `lenet5`, `alexnet`)
1. Follow the project structure conventions
1. Include comprehensive documentation
1. Write thorough tests
1. Add examples demonstrating usage
